# RibbonMacroTest

Демонстрационный проект для проверки механизма привязки макрофайлов (.mac) к командам Ribbon.

## Что демонстрирует

- Главное окно на базе `SARibbonMainWindow` с несколькими категориями и панелями Ribbon.
- `ApplicationWidgetBase` с новой вкладкой "Команды", открывающейся по кнопке "Файл".
- Редактор макрокоманд: добавление/удаление макросов, изменение названия и иконки.
- Виджет настройки панелей Ribbon (`SARibbonCustomizeWidget`) для размещения команд на ленте.
- Сохранение и загрузка конфигурации команд и панелей в `RibbonMacroTest.ini`.

## Как использовать

1. Соберите проект `ToolsRuntimeProj.pro` (или `RibbonMacroTest.pro` вручную).
2. Запустите `RibbonMacroTest.exe`.
3. Нажмите кнопку **Файл** в левом верхнем углу — откроется `ApplicationWidgetBase`.
4. Перейдите на вкладку **Команды**.
5. На первой странице "Команды" добавьте макрос `.mac` (например, `hello.mac` из корня проекта).
6. Перейдите на вторую страницу "Настройка ленты" и перетащите макрокоманду на панель Ribbon.
7. Закройте `ApplicationWidgetBase` — настройки сохранятся автоматически.
8. Нажмите созданную кнопку на Ribbon — выполнится макрос.

## Требования к макросам

Макрос должен содержать процедуру `ExecAction`. Пример:

```rsl
Процедура ExecAction()
    Сообщить("Привет из RibbonMacroTest!");
КонецПроцедуры
```

## Файлы

- `mainwindow.h/cpp` — главное окно с Ribbon и `ApplicationWidgetBase`.
- `main.cpp` — точка входа.
- `hello.mac` — пример макроса.
